# AI-Capstone

**Instructor: Prof. Tsai-Pei Wang**

This repository contains the coursework and final project for the AI-Capstone course, focusing on the process of constructing and analyzing a dataset, Reinforcement Learning, and Computer Vision.

---

## Course Modules & Summaries

### Project 1: Music Genre Classification & Machine Learning Analysis
This assignment explores audio signal processing and the application of supervised/unsupervised learning techniques.
* **Data Processing**: Extracted features from YouTube audio tracks using **MFCC** (Mel-Frequency Cepstral Coefficients).
* **Supervised Learning**: Evaluated the impact of training set size, **Class Weighting**, and **PCA** (Principal Component Analysis) on model performance.
* **Unsupervised Learning**: Explored clustering quality by analyzing the **Adjusted Rand Score** across varying cluster counts.

### Project 2: Reinforcement Learning (Q-Learning & DQN)
Implementation of classic Reinforcement Learning algorithms using **PyTorch**.
* **Core Methodology**: Developed a **Deep Q-Network (DQN)** combining Q-Learning theory with neural network function approximation.
* **Stability Mechanisms**: Integrated **Experience Replay** and **Target Networks** to stabilize the training process in high-dimensional state spaces.
* **Environments**: Benchmarked performance using **OpenAI Gym** interfaces, specifically on `Blackjack-v1` and `ALE/Assault-v5`.

### Final Project: Traffic Accident Detection
A Computer Vision-based project focused on understanding dynamic road scenes and identifying collisions.
* **Motivation**: Improving public safety by automating accident detection to decrease emergency response times.
* **Technical Stack**: Leveraged Deep Learning models (such as **YOLO**) for real-time Object Detection and scene understanding.
* **Applications**: Designed for Smart City infrastructure, traffic monitoring systems, and autonomous vehicle safety protocols.
